# PetSitters

## Описание

Сервис предоставляет возможность заказа услуги на выгул и уход за животными.

## Типы пользователей

**Клиент** — может регистрировать своих животных в системе и выставлять заявки на выгул или уход.

**Петситтер** — может выполнять заявки, оставленные клиентами.

## Функциональность

- Регистрация и аутентификация клиентов и петситтеров
- Регистрация животных (CRUD операции)
- Создание и выполнение заявок (полный workflow)
- Система отзывов для петситтеров (с автообновлением рейтинга)

## Настройка окружения разработки

Создать файл в корне репозитория `/local.env`, используя в качестве примера файл `/local.env.sample`

## Запуск приложения
- Development-mode: 
    ```bash
    docker compose -f ./docker-compose.local.yaml --env-file ./local.env up -d --wait
    ```
- Production-mode:
    ```bash
    docker compose --env-file ./local.env up -d --wait
    ```

После запуска приложение будет доступно:
- **API:** http://localhost:3110
- **Swagger документация:** http://localhost:3110/api
- **Mongo Express:** http://localhost:8081 (admin/admin123)
